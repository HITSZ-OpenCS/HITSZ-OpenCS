### 关于考试

作为一门很纯正的数学类科目，基本上是「会者不难，难者不会」。不过捞人还是捞的。

考试的题目类型很固定，看几份卷子就能总结出来。全是大题，基本不存在死扣概念的题目，所以理解比背记重要得多。
